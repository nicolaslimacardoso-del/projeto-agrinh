## 📖 Sobre o Projeto

O projeto apresenta a relação entre o campo e a cidade, destacando a importância da agricultura para a sociedade e como a tecnologia contribui para aproximar produtores e consumidores.

A página foi desenvolvida utilizando HTML, CSS e JavaScript, com foco em uma interface moderna, responsiva e interativa.

---

## 🎯 Objetivo

Demonstrar a conexão entre o campo e a cidade por meio de informações sobre:

- Sustentabilidade na agricultura;
- Uso da tecnologia no campo;
- Importância da produção agrícola para a população urbana;
- Valorização do trabalho realizado pelos produtores rurais.

---

## ✨ Funcionalidades

- Layout responsivo para diferentes tamanhos de tela;
- Navegação por seções;
- Modo escuro (Dark Mode);
- Galeria de imagens;
- Contador animado;
- Rolagem suave (Smooth Scroll);
- Formulário de contato para demonstração visual.

## ▶ Como Executar

1. Baixe ou clone o projeto.
2. Certifique-se de que as imagens estejam na pasta `img`.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

Não é necessário instalar dependências ou utilizar servidor.
