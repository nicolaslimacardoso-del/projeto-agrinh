# Projeto Agrinho 2026

## Conexão Campo e Cidade

Este projeto foi desenvolvido para o Programa Agrinho 2026 com o tema **Conexão Campo e Cidade**.

### Objetivo

Demonstrar como a tecnologia aproxima o campo e a cidade, destacando:

- Sustentabilidade na agricultura;
- Uso de tecnologias no campo;
- Importância da produção agrícola para a população urbana.

### Funcionalidades

- Layout responsivo;
- Modo escuro (Dark Mode);
- Galeria de imagens;
- Contador animado;
- Scroll suave entre seções;
- Formulário de contato.

### Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
